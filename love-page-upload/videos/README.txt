Put timeline videos here, preferably as compressed MP4 files.

Recommended usage in index.html:

media: [
  { type: 'image', src: 'photos/example.jpg' },
  { type: 'video', src: 'videos/example.mp4', poster: 'photos/example-poster.jpg' }
]

Keep videos reasonably small for GitHub Pages. MP4 with H.264 video is the safest choice for phones.
